# HTML

Lenguaje de Marcas de Hipertexto [(**H**yper **T**ext **M**arkup **L**anguage)](https://developer.mozilla.org/es/docs/Web/HTML) es el estándar básico utilizado para la estructuración de páginas web. Fue introducido por el World Wide Web Consortium [W3C](https://es.wikipedia.org/wiki/W3C) en 1991. La versión vigente es HTML5 y fué introducida en 2006.

*Hipertexto* hace referencia a los vínculos que conectan páginas web, ya sea a otros sitios web o a un sitio diferente dentro del mismo. Este concepto es el concepto básico del internet.

El HTML se basa en etiquetas, estas etiquetas se utilizan para definir distintos elementos que luego el navegador web interpreta, como `<head>`, `<body>`, entre muchas otras.
