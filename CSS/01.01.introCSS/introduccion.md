# CSS

Hojas de Estilo en Cascada [(Cascading Style Sheets)](https://developer.mozilla.org/es/docs/Web/CSS) es un lenguaje estadarizaod que se utiliza para manejar la presentación de documentos HTML o XML y sus derivados. CSS contiene una serie de reglas que el navegador web utiliza para renderizar los estilos de los documentos. 

El estándar es regulado por la [W3C](https://www.w3.org/Style/CSS/#specs). Se introdujo en 1994 para facilitar los retos que hasta ese momento suponía manejar la presentación de documentos directamente en HTML.

El propósito de CSS es separar la estilización de documentos de su definición en HTML. Esto permite que el manejo de estilos sea más sencillo y adaptable.

## Jerarquía de Cascada

El concepto de Cascada en CSS hace referencia a la preferencia que toma la definición de estilos, es decir, cuáles estilos se aplican en ausencia de otros. A continuación se lista esta jerarquía en orden de priorización:

1. *Inline Style*, el estilo definido directamente en el elemento.
2. *Embedded Page Styles*, los estilos definidos para la página, en el encabezado del HTML.
3. Estilos Externos, los estilos definidos en un archivo CSS anclado al HTML.
4. Estilos definidos por default por el navegador web.
